## macos 安装 riscv 调试工具

参考：<https://pdos.csail.mit.edu/6.828/2022/tools.html>

```shell
brew tap riscv/riscv
brew install riscv-tools
```

安装后测试

```shell
riscv64-unknown-elf-gdb --version
```
